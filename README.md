# 🚀 CUDA Learning Journey (PMPP-Based)

A structured, chapter-by-chapter documentation of my GPU programming journey using **CUDA C++**, based on the book *Programming Massively Parallel Processors (PMPP)* by Kirk & Hwu.

---

## 🎯 Goals

* Understand GPU architecture and parallel programming fundamentals
* Learn CUDA programming from the ground up
* Analyse performance with NVIDIA Nsight Systems & Nsight Compute
* Build a professional HPC portfolio on GitHub

---

## 📂 Repository Structure

```
cuda-pmpp-learning/
├── 01_vector_add/          # Chapter 1 – Hello CUDA (vector addition)
├── 02_memory_management/   # Chapter 2 – Host/device memory model
├── notes/                  # Concept notes and theory summaries
├── scripts/                # Reusable build, run, and profiling scripts
└── assets/                 # Diagrams and reference images
```

| Folder                   | Description                                          |
| ------------------------ | ---------------------------------------------------- |
| `01_vector_add/`         | First CUDA kernel – basic parallel vector addition   |
| `02_memory_management/`  | Host ↔ device transfers, unified memory, pinned mem  |
| `notes/`                 | Markdown notes on CUDA concepts and GPU architecture |
| `scripts/`               | Shell scripts for build, run, and profiling          |
| `assets/`                | Architecture diagrams and profiling screenshots      |
